# 비즈니스 요구사항과 설계

## 회원 도메인 활용 관계

```plaintext
클라이언트 -> 회원 서비스(회원가입, 회원조회) -> 회원 저장소
                               |
          -------------------------------------------------
          |                      |                        |
        메모리                    DB                   외부 시스템 연동
        회원 저장소              회원 저장소                 회원 저장소
```
